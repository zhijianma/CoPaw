# CoPaw 与 OpenClaw 的功能对比

| 分类     | 维度            | OpenClaw                                                                                       | CoPaw                                                                                                                                                                       |
| :------- | :-------------- | :--------------------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 技术栈   | 主要语言        | TypeScript / Node.js                                                                           | Python                                                                                                                                                                      |
|          | 底层智能体框架  | 深度集成 pi-agent SDK 作为核心智能体引擎                                                       | • AgentScope 多智能体框架<br>• AgentScope-Runtime 智能体运行时                                                                                                              |
|          | 记忆系统        | 文件式记忆 (JSONL+MD)                                                                          | File Based 记忆系统 Powered by AgentScope-AI/ReMe                                                                                                                           |
| 使用体验 | 安装方式        | • npm install -g openclaw<br>• openclaw onboard CLI 向导                                       | • 一行脚本安装<br>• 备选 pip install copaw<br>• docker 安装                                                                                                                 |
|          | 平台支持        | macOS / Linux / Windows (WSL2)                                                                 | macOS / Linux / Windows（PowerShell/CMD）                                                                                                                                   |
|          | 自托管/云端部署 | Docker 自托管                                                                                  | • Docker 自托管（agentscope/copaw:latest，含 Web Console）<br>• 一键云端部署                                                                                                |
|          | 本地模型支持    | 通过配置文件指定 Ollama / llama.cpp 端点                                                       | 安装时通过 --extras 参数按需集成：<br>• --extras llamacpp（跨平台）<br>• --extras mlx（Apple Silicon）                                                                      |
|          | Skills支持      | • 支持本地 skills<br>• 从 clawhub install                                                      | • 支持本地 skills<br>• 直接从多个社区 Skills Hub 导入 Skills<br>（现支持：https://skills.sh/...，https://clawhub.ai/...，https://skillsmp.com/...，https://github.com/...） |
|          | 频道接入        | WhatsApp, Telegram, Slack, Discord, iMessage, Signal, Teams, Matrix, Zalo 等                   | 钉钉、飞书、QQ、Discord、iMessage 等，适配国内生态，支持channel扩展                                                                                                         |
| 社区生态 | 开源协议        | MIT (完全开放)                                                                                 | Apache 2.0 (商业友好)                                                                                                                                                       |
|          | 可扩展性        | 支持社区技能接入                                                                               | 支持社区技能接入                                                                                                                                                            |
| Roadmap  | 安全模型        | • 社区驱动的安全加固 (WASM 沙箱，API Key 保护等)<br>• ClawHub 技能市场全量接入 VirusTotal 扫描 | • 社区驱动<br>• 官方 Roadmap: 工具执行沙箱                                                                                                                                  |
|          | Agent 协同      | 社区驱动                                                                                       | 基于AgentScope，原生支持多智能体                                                                                                                                            |
|          | 云原生扩展      | 社区驱动                                                                                       | • 社区驱动<br>• 官方 Roadmap: 结合 AgentScope-Runtime 向云端算力、存储与生态持续延伸                                                                                        |
|          | 大小模型协同    | 官方文档未明确声明模型策略                                                                     | 官方 Roadmap: 大小模型协同，轻量本地模型处理隐私数据，云端大模型负责规划等复杂任务, 兼顾安全、性能、与能力                                                                  |
|          | 本地模型训练    | 社区驱动                                                                                       | 官方 Roadmap: 针对核心工作流训练小参数量本地模型，提升可用性                                                                                                                |
|          | 多模态交互      | 社区驱动                                                                                       | 官方 Roadmap: 支持进行语音和视频形式的互动                                                                                                                                  |
|          | 社区生态        | 社区驱动，ClawHub 持续扩展                                                                     | 官方 Roadmap: 持续丰富 AgentScope-Skills 仓库，提升优质Skill的可发现性                                                                                                      |
