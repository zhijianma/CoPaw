# CoPaw vs OpenClaw: Feature Comparison

| Category            | Dimension                       | OpenClaw                                                                                                                                        | CoPaw                                                                                                                                                                                                    |
| :------------------ | :------------------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tech Stack          | Primary Language                | TypeScript / Node.js                                                                                                                            | Python                                                                                                                                                                                                   |
|                     | Agent Framework                 | Deeply integrated with pi-agent SDK as the core agent engine                                                                                    | • AgentScope multi-agent framework<br>• AgentScope-Runtime                                                                                                                                               |
|                     | Memory System                   | File-based memory (JSONL + MD)                                                                                                                  | File-based memory system powered by AgentScope-AI/ReMe                                                                                                                                                   |
| User Experience     | Installation                    | • npm install -g openclaw<br>• openclaw onboard CLI wizard                                                                                      | • One-line script installation<br>• Alternative: pip install copaw<br>• Docker installation                                                                                                              |
|                     | Supported Platform              | macOS / Linux / Windows (WSL2)                                                                                                                  | macOS / Linux / Windows (PowerShell/CMD)                                                                                                                                                                 |
|                     | Self-hosting / Cloud Deployment | Self-hosted via Docker                                                                                                                          | • Self-hosted via Docker (agentscope/copaw:latest, with Web Console)<br>• One-click cloud deployment                                                                                                     |
|                     | Local Model Support             | Configure Ollama / llama.cpp endpoints via config file                                                                                          | On-demand integration via --extras during installation:<br>• --extras llamacpp (cross-platform)<br>• --extras mlx (Apple Silicon)                                                                        |
|                     | Skills Support                  | • Supports local skills<br>• Installation from ClawHub                                                                                          | • Supports local skills<br>• Direct import from multiple community Skills Hubs<br>(Currently supported: https://skills.sh/..., https://clawhub.ai/..., https://skillsmp.com/..., https://github.com/...) |
|                     | Channel Integrations            | WhatsApp, Telegram, Slack, Discord, iMessage, Signal, Teams, Matrix, Zalo, etc.; extensible channel support                                     | DingTalk, Feishu, QQ, Discord, iMessage, etc.; extensible channel support                                                                                                                                |
| Community Ecosystem | Open-source License             | MIT (fully open)                                                                                                                                | Apache 2.0 (business-friendly)                                                                                                                                                                           |
|                     | Extensibility                   | Community Skills integrations supported                                                                                                         | Community Skills integrations supported                                                                                                                                                                  |
| Roadmap             | Security Model                  | • Community-driven security hardening (WASM sandbox proposals, API key protection, etc.)<br>• ClawHub marketplace with full VirusTotal scanning | • Community-driven<br>• Official roadmap: tool-sandbox                                                                                                                                                   |
|                     | Agent Collaboration             | Community-driven                                                                                                                                | Inherits native multi-agent support through AgentScope framework                                                                                                                                         |
|                     | Cloud-Native Architecture       | Community-driven                                                                                                                                | • Community-driven<br>• Official roadmap: continuously extend cloud capabilities via AgentScope Runtime, covering cloud compute, storage, and a broad cloud service ecosystem                            |
|                     | Large-Small Model Collaboration | Not explicitly stated in official docs                                                                                                          | Official roadmap: lightweight local LLMs securely handle privacy-sensitive data; complex tasks (e.g. planning, coding) routed to powerful cloud LLMs — balancing security, performance, and capability   |
|                     | Local Model Training            | Community-driven                                                                                                                                | Official roadmap: train and fine-tune locally deployable LLMs specifically for CoPaw's core native Skills and common tasks                                                                               |
|                     | Multimodal Interaction          | Community-driven                                                                                                                                | Official roadmap: voice and video interaction support                                                                                                                                                    |
|                     | Community Ecosystem             | Community-driven; ClawHub continuously expanding                                                                                                | Official roadmap: continuously enrich the AgentScope-Skills repository and improve the discoverability of high-quality Skills                                                                            |
